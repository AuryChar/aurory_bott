## 🛠️ Instalação e Execução  
1. Clone o repositório:  
   ```bash
   git clone https://github.com/AuryChar/aurory-bott.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure seu .ENV com seus tokens:
   ```env
   CLIENT_ID=seu_token
   TOKEN=seu_token
   ```
4. Execute o bot:
   ```bash
   node index.js
   ```
